# 通过电脑 USB 辅助安装开途 Android 版

## 适用场景

在手机上多次尝试仍然无法安装开途时，可以通过电脑帮手机完成安装。这种方式可以绕过手机上的所有安装限制。

## 前提条件

- 一台 **Windows 或 Mac 电脑**
- 电脑上已安装**开途桌面版**（可从 kaitu.io/install 下载）
- 一根 **USB 数据线**（必须是能传输数据的线，不能是只能充电的线）

## 安装步骤

### 第一步：手机开启"开发者选项"和"USB 调试"

1. 打开手机 **"设置"** → **"关于手机"**
2. 找到 **"版本号"**，连续点击 **7 次**
3. 屏幕会提示 **"您已进入开发者模式"**
4. 返回 **"设置"** 主页，找到 **"开发者选项"**（有些手机在"系统"或"其他设置"下面）
5. 进入"开发者选项"，找到并打开 **"USB 调试"** 开关

**小米手机特别注意**：除了打开"USB 调试"外，还需要找到 **"USB 调试（安全设置）"** 这个选项并**关闭**它，否则电脑可能检测不到手机。

### 第二步：连接手机和电脑

1. 用 USB 数据线将手机连接到电脑
2. 手机上会弹出 **"允许 USB 调试吗？"** 的提示
3. 建议勾选 **"始终允许来自这台计算机的调试"**，然后点击 **"允许"**

### 第三步：在桌面版中安装

1. 在电脑上打开**开途桌面版**
2. 进入 **"账户"** → **"其他设备安装"** → **"安卓手机安装"**
3. 桌面版会自动检测到你的手机，并显示设备名称
4. 点击 **"开始安装"**，等待安装完成（通常需要 1-2 分钟）
5. 安装完成后，手机上会出现开途应用图标

### 安装进度说明

安装过程中，桌面版会依次显示以下状态：

- **正在下载应用** → 从服务器获取最新版安装包
- **正在传输到手机** → 将安装包通过 USB 发送到手机
- **正在安装到手机** → 在手机上完成安装
- **安装成功** → 可以拔掉数据线，在手机上打开开途了

## 常见问题

### 检测不到手机

- 确认使用的是**数据线**而不是充电线。简单测试方法：连接后看电脑上能不能浏览手机里的文件，如果不能，多半是充电线。
- 确认手机上的 **"USB 调试"** 已经打开。
- 检查手机屏幕上是否有弹窗要求授权，如果有请点击"允许"。
- 尝试换一个 USB 接口，或者换一根数据线。

### Windows 电脑检测不到手机

- 可能需要安装手机品牌对应的 USB 驱动程序。
- 尝试重启电脑后再试。
- 检查设备管理器中是否有带黄色感叹号的未知设备。

### 小米手机检测不到

- 确认已经**关闭**了 **"USB 调试（安全设置）"** 这个选项。注意是关闭，不是打开。这个选项和"USB 调试"是两个不同的开关。

### 安装失败提示"存储空间不足"

- 手机存储空间不够了，请清理一些不需要的文件、照片或应用后重试。

### 安装完成后需要关闭"USB 调试"吗？

- 建议安装完成后回到"开发者选项"，关闭"USB 调试"。日常使用不需要开着这个选项。
