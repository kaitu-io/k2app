# iOS 安装指南（iPhone / iPad）

## 支持设备

开途支持所有 iPhone 和 iPad 设备。

## 下载安装

1. 打开手机上的 **App Store**
2. 在搜索栏中输入 **"Kaitu"** 或 **"开途"**
3. 找到开途应用，点击 **"获取"**
4. 使用 Face ID、Touch ID 或 Apple ID 密码确认下载
5. 等待下载安装完成

## 登录和使用

1. 在主屏幕上找到开途图标，点击打开
2. 输入您的注册邮箱
3. 点击 **"发送验证码"**
4. 前往邮箱查收验证码（注意检查垃圾邮件文件夹）
5. 输入验证码，完成登录

## 首次连接 VPN 授权

首次点击连接时，系统会弹出安全提示：

1. 屏幕上出现提示 **"'Kaitu' 想要添加 VPN 配置"**
2. 点击 **"允许"**
3. 系统要求验证身份：输入手机锁屏密码，或使用 Face ID / Touch ID 确认
4. VPN 配置添加成功，自动开始连接
5. 连接成功后，屏幕顶部状态栏会出现 VPN 图标

> 这是 iOS 系统的标准安全流程，所有 VPN 类应用首次使用时都需要此步骤，不是风险提示，请放心点击"允许"。此操作只需要在首次连接时进行一次。

## 在 App Store 搜不到开途怎么办？

如果在 App Store 中搜索不到开途，可能是以下原因：

### 检查 Apple ID 地区

开途可能在部分地区的 App Store 尚未上架。请确认您的 Apple ID 所在地区：

1. 打开 **"设置"**
2. 点击顶部的 **您的姓名**
3. 点击 **"媒体与购买项目"**
4. 查看当前地区设置

### 通过链接直接访问

尝试在 Safari 浏览器中直接访问开途的 App Store 页面链接，跳过搜索步骤。您可以在 [kaitu.io/install](https://kaitu.io/install) 页面找到 App Store 下载链接。

### 联系客服

如果以上方法都无法解决，请联系开途客服获取帮助，我们会为您提供其他安装方式。
