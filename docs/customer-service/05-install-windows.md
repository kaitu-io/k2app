# Windows 安装指南

## 系统要求

开途支持 Windows 10 和 Windows 11 系统。

## 下载开途

1. 在浏览器中访问 [kaitu.io/install](https://kaitu.io/install)
2. 网页会自动识别您的操作系统，5 秒后自动开始下载
3. 如果没有自动下载，请点击页面上的下载按钮手动下载
4. 下载的文件名格式为：`Kaitu_版本号_x64.exe`

## 安装步骤

1. 找到下载好的 `Kaitu_版本号_x64.exe` 文件（通常在"下载"文件夹中）
2. 双击运行安装程序
3. 按照安装向导的提示完成安装
4. 安装完成后，开途会自动启动
5. 使用您的账号登录，即可开始使用

## 遇到 Windows SmartScreen 拦截怎么办？

安装时，您可能会看到一个蓝色弹窗，提示 **"Windows 已保护你的电脑"**（英文系统显示为 "Windows protected your PC"）。这是 Windows 系统对新软件的安全检查，不代表软件有问题。

处理方法：

1. 在弹窗中，点击 **"更多信息"**（英文为 "More info"）
2. 页面会展开显示应用信息
3. 点击 **"仍要运行"**（英文为 "Run anyway"）
4. 安装程序就会正常启动

> 出现这个提示是因为开途是较新的软件，尚未积累足够的 Windows 信誉评分。开途 Windows 版已通过 EV 代码签名认证，软件本身是安全的，请放心安装。

## 遇到浏览器下载拦截怎么办？

部分浏览器在下载时可能会弹出安全提示，以下是常见浏览器的处理方法：

### Chrome 浏览器

1. 下载时可能提示 **"xxx 不常见，可能存在危险"**
2. 点击提示右侧的 **"^"** 箭头图标
3. 选择 **"保留"**
4. 文件即可正常下载

### Edge 浏览器

1. 下载时可能弹出类似的安全提示
2. 点击 **"保留"**
3. 如果继续提示，再点击 **"仍然保留"**
4. 文件即可正常下载

## 安全说明

开途 Windows 版已通过 EV（Extended Validation）代码签名认证，这是 Windows 平台最高级别的软件签名认证。浏览器和系统的安全提示是对所有新软件的常规检查流程，并非针对开途，请放心使用。
