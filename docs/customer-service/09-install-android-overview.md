# Android 安装总览

## 系统要求

开途支持 Android 7.0 及以上版本的手机和平板。

开途 Android 版目前通过直接下载 APK 安装包的方式安装，暂未上架 Google Play 商店。

## 两种安装方式

### 方式一：手机直接下载安装（推荐优先尝试）

1. 用手机自带浏览器打开 **kaitu.io/install**
2. 点击页面上的下载按钮，等待下载完成
3. 下载完成后点击安装包进行安装
4. 安装过程中可能会出现安全提示或风险警告，需要手动点击"允许"或"继续安装"

不同品牌的手机弹出的提示不一样，具体操作请参考对应品牌的安装指南（华为、小米、OPPO、vivo 等）。

### 方式二：通过电脑 USB 辅助安装（手机上实在装不了时使用）

如果在手机上多次尝试都无法完成安装，可以借助电脑来安装：

1. 在 Windows 或 Mac 电脑上打开**开途桌面版**
2. 用 USB 数据线将手机连接到电脑
3. 在桌面版中进入 **"账户"** → **"其他设备安装"** → **"安卓手机安装"**
4. 桌面版会自动检测到你的手机，并帮你把应用安装到手机上

这种方式可以完全绕过手机上的各种安装限制，安装成功率最高。

## 为什么 Android 安装会遇到阻拦？

国产手机系统（华为、小米、OPPO、vivo 等）内置了多层应用安全检查机制，包括：

- **纯净模式**：限制安装非官方应用商店的应用
- **云安全扫描**：将安装包上传云端检测
- **应用备案检查**：检查应用是否有国内备案
- **病毒扫描提示**：提示应用存在风险

这些拦截是因为应用不是从手机自带应用商店安装的，并不代表应用本身有任何安全问题。开途应用经过安全签名认证，可以放心安装使用。

## 安装后首次打开

- 首次打开开途时，系统会弹出 **VPN 连接权限请求**，请点击 **"允许"** 或 **"确定"**。这是 Android 系统的标准权限请求，开途需要此权限才能正常工作。
