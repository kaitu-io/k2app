# Android 各品牌手机安装指南

不同品牌的安卓手机在安装非应用商店下载的应用时，会弹出不同的安全提示和拦截。以下是各品牌的详细绕过方法。

> 为什么会被拦截？国产手机系统内置了多层安全检查（纯净模式、云安全扫描、应用备案检查等），这些拦截是因为应用不是从手机自带商店安装的，不代表应用有问题。开途应用经过安全签名认证，可以放心安装。

---

## 华为 / HUAWEI

### HarmonyOS 4.2 及更早版本（支持安装）

华为的「纯净模式」中的「增强防护」是拦截安装的核心机制。需要先关闭它：

**关闭增强防护：**
1. 打开 **「设置」**
2. 向下滑动，进入 **「系统和更新」**
3. 找到并点击 **「纯净模式」**
4. 找到 **「增强防护」** 选项，点击将其 **关闭**
5. 返回文件管理器，重新点击 APK 文件安装

安装完成后，建议返回设置重新开启「增强防护」，保持日常安全。

**如果安装时仍弹出风险提示：**
- 点击提示框中的 **「仍要安装」** 或 **「继续安装」**

### HarmonyOS NEXT（5.0 及以上）— 无法安装 APK

**重要提示：鸿蒙 NEXT 系统已彻底移除了安卓兼容层，APK 文件完全无法安装和运行。** 这不是设置问题，是系统底层不再支持。

如果您的华为手机已升级到 HarmonyOS NEXT（通常显示为 HarmonyOS 5.0、5.1 或更高版本）：
- 目前无法在该设备上使用开途 Android 版
- 请关注开途官方动态，我们正在评估鸿蒙原生版本的开发计划
- 如需使用开途，建议暂时使用其他设备（电脑、iPhone 或未升级鸿蒙 NEXT 的手机）

**如何查看系统版本：** 设置 → 关于手机 → 查看 HarmonyOS 版本号

## 小米 / Xiaomi（含红米 Redmi）

小米的拦截分为两个阶段：**下载时浏览器拦截** + **安装时安全守护拦截**。

### 第一步：解决浏览器下载拦截

小米自带浏览器可能直接阻止下载，提示"该站点包含违规内容"或跳转到应用商店。

**解决方法（任选一种）：**
- **方法 A（推荐）：** 改用 Chrome 或 Edge 浏览器下载，不要用小米自带浏览器
- **方法 B：** 在自带浏览器中进入 设置 → 隐私防护（或安全设置）→ 关闭 **「安全网址拦截」**

### 第二步：解决安装拦截

#### HyperOS（澎湃OS）1.0 及以上

安装时会出现大面积红色警告界面，看不到安装按钮。

1. 在红色警告界面，点击右上角的 **齿轮图标**（设置）
2. 进入 **「安全守护」** 管理界面
3. 点击 **「关闭安全守护」**
4. 系统会要求输入 **小米账号密码** 验证身份，输入后确认
5. 返回安装界面，等待倒计时结束后，点击 **「无视风险安装」** 或 **「单次允许安装」**

#### MIUI 14

1. 打开 **「设置」** → **「应用设置」** → **「纯净模式」**
2. 选择 **「关闭纯净模式」**，确认退出
3. 重新点击 APK 安装
4. 勾选 **「我已了解风险」**
5. 等待 **10 秒倒计时** 结束后，点击 **「仍要安装」**

#### 极端情况：以上方法都不行

如果被小米云端特征库标记为"极度危险"，即使关闭安全守护仍然被拦截，唯一方法是通过电脑 USB 辅助安装（ADB 方式），详见「通过电脑 USB 辅助安装」指南。

## OPPO / 一加 OnePlus（ColorOS）

ColorOS 的特点是即使授予了权限，联网状态下仍可能被云端策略拦截。

### 标准流程

1. 打开 **「设置」** → **「密码与安全」** → **「系统安全」**
2. 找到 **「外部来源应用」**
3. 找到你用来下载的浏览器（如 Chrome），开启 **「允许安装应用」**
4. 返回点击 APK 文件安装

### 如果联网状态下仍被拦截（断网安装法）

ColorOS 的云安全引擎在断网后会自动降级为宽松模式，这是最有效的绕过方法：

1. 遇到拦截提示后，**先退出安装界面**
2. 开启 **「飞行模式」**（从屏幕顶部下拉，点击飞行模式图标）
3. 打开手机的多任务界面，**彻底关闭「软件商店」和「手机管家」**（向上滑掉这两个应用）
4. 在飞行模式下，用文件管理器重新点击 APK 安装
5. 此时只会弹出常规提示，点击 **「无视风险安装」** 即可
6. 安装完成后关闭飞行模式

## vivo / iQOO（OriginOS）

OriginOS 的安装界面会故意隐藏「继续安装」按钮，同时突出显示「删除安装包」来引导你放弃安装。

### 标准流程

1. 打开 **「设置」** → **「安全与隐私」** → **「更多安全设置」**
2. 找到 **「应用安装」** 模块
3. 找到你用来下载的应用（如浏览器、QQ、微信），开启其安装权限
4. 返回点击 APK 安装

### 安装界面找不到继续按钮时

vivo 经常在警告界面隐藏安装入口，试试以下方法：

- 在弹出警告框时，点击右上角的 **「详细信息」** 或 **「更多」** 文字
- 在展开的详细描述中，寻找被弱化显示的 **「继续安装」** 入口
- 如果上述入口都找不到，尝试使用断网安装法（参考 OPPO 的飞行模式方法）

### 极端情况：文件管理器拒绝打开 APK

如果 vivo 自带文件管理器直接拒绝识别 APK 文件：
1. 将下载的文件后缀从 `.apk` 改为 `.zip`
2. 使用第三方文件管理器（如 MT 管理器）打开该文件
3. 在第三方管理器中将后缀改回 `.apk`
4. 在第三方管理器中直接点击安装

## 三星 / Samsung（One UI）

三星在 One UI 6.0 及以上引入了「自动拦截程序」(Auto Blocker)，会全屏阻止安装。

### One UI 6.1.1 及以上

1. 打开 **「设置」**
2. 向下滚动，点击 **「安全和隐私」**
3. 找到 **「自动拦截程序」**（Auto Blocker）
4. 点击顶部的开关，将其 **关闭**
5. 系统会要求**指纹/面部识别或密码**验证身份
6. 验证通过后，返回点击 APK 安装

### One UI 6.1 及更早版本

1. 打开 **「设置」** → **「安全与隐私」**
2. 滑动到最底部，点击 **「更多安全设置」**
3. 找到 **「自动拦截程序」**，将其关闭
4. 返回上一级，进入 **「安装未知应用」**
5. 找到你用的浏览器，开启 **「允许来自此来源」**

## 荣耀 / HONOR（MagicOS）

荣耀的「纯净模式」与华为类似但独立开发，拦截界面会隐藏安装按钮或显示为灰色。

### MagicOS 7.x — 8.0

1. 打开 **「系统设置」**
2. 向下滑动，点击 **「系统和更新」**
3. 点击带有绿色盾牌标识的 **「纯净模式」**
4. 在界面底部找到 **「退出」** 按钮（比较小，注意找）
5. 系统弹出警告，坚决点击 **「仍要退出」**
6. 退出纯净模式后，重新点击 APK 安装

安装完成后建议重新进入纯净模式。

---

## 所有品牌通用：VPN 权限弹窗

无论什么品牌，首次打开开途连接 VPN 时，系统都会弹出权限确认：

- 提示内容类似"该应用将接管网络流量"或"允许建立 VPN 连接"
- **必须点击「允许」或「确定」**，否则 VPN 无法工作
- 如果不小心点了拒绝，需要去 设置 → 应用管理 → 开途 → 权限，重新授予网络权限

## 实在装不上怎么办？

如果按照以上指南操作后仍然无法安装，推荐使用 **电脑 USB 辅助安装**（ADB 方式）。通过电脑直接推送到手机，可以绕过手机上的所有安装限制。详见「通过电脑 USB 辅助安装」指南。
