# macOS 安装指南

## 系统要求

开途支持 macOS 12 (Monterey) 及以上版本，Intel 芯片和 Apple Silicon 芯片（M1/M2/M3/M4）均可使用。

## 下载开途

1. 在浏览器中访问 [kaitu.io/install](https://kaitu.io/install)
2. 网页会自动识别您的操作系统，5 秒后自动开始下载
3. 如果没有自动下载，请点击页面上的下载按钮手动下载
4. 下载的文件名格式为：`Kaitu_版本号_universal.pkg`

## 安装步骤

1. 找到下载好的 `Kaitu_版本号_universal.pkg` 文件（通常在"下载"文件夹中）
2. 双击打开安装包
3. 按照安装向导的提示，点击 **"继续"** 和 **"安装"** 完成安装
4. 安装过程中可能需要输入您的 Mac 登录密码来确认
5. 安装完成后，在启动台（Launchpad）或应用程序文件夹中找到开途，点击打开
6. 使用您的账号登录，即可开始使用

## 遇到 macOS 安全提示怎么办？

如果双击安装包后弹出提示 **"无法验证开发者"**（英文为 "cannot verify the developer"），请使用以下任一方法解决：

### 方法一：右键打开（推荐）

1. 在 Finder 中找到下载的安装包文件
2. **右键点击**（或按住 Control 键点击）安装包
3. 在弹出的菜单中选择 **"打开"**
4. 在弹出的确认窗口中，再次点击 **"打开"**
5. 安装程序就会正常启动

### 方法二：通过系统设置允许

1. 打开 **"系统设置"**（旧版 macOS 为"系统偏好设置"）
2. 点击 **"隐私与安全性"**
3. 向下滚动，在页面下方会看到被阻止的应用提示
4. 点击 **"仍要打开"**
5. 输入 Mac 登录密码确认
6. 安装程序就会正常启动

> 开途 macOS 版已通过 Apple 官方公证（Notarization），软件是安全的。出现此提示是 macOS 的标准安全保护机制，请放心安装。

## 首次使用 VPN 配置授权

首次连接时，macOS 会弹出提示，要求安装 VPN 配置：

1. 系统弹出 **"开途想要添加 VPN 配置"** 的提示
2. 点击 **"允许"**
3. 输入您的 Mac 登录密码，或使用 Touch ID 确认
4. VPN 配置安装完成，即可正常连接

这是 macOS 系统的标准安全流程，只需要在首次使用时操作一次。
