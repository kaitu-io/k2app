# 常见问题排查

## 连接问题

### 连接后无法上网

1. **切换路由模式**：在主界面尝试切换"全局模式"和"智能模式"，切换后会自动重连。
2. **断开重连**：点击断开，等几秒后重新连接。
3. **检查基础网络**：关闭开途后看看能不能正常上网。如果关闭开途后也上不了网，说明是网络本身的问题，不是开途的问题。
4. **重启应用**：完全关闭开途再重新打开。
5. **尝试其他节点**：当前节点可能临时出现问题，换一个节点试试。

### 连接缓慢或频繁断开

- 当前网络环境可能不太稳定，开途会自动尝试恢复连接，请稍等片刻。
- 可以尝试切换到其他节点，选择延迟更低的节点通常体验更好。
- 确保手机或电脑上没有同时运行其他 VPN 软件，多个 VPN 同时运行会互相冲突。
- 手机用户请确认开途已加入电池优化白名单，否则系统可能在后台关闭开途。

### 显示"连接失败"

1. 检查手机或电脑的网络连接是否正常。
2. 检查开途订阅是否已过期（进入"账户"页面查看）。
3. 重启应用后重试。
4. 如果持续连接失败，请通过应用内反馈功能提交反馈（会自动附带日志），或联系客服获取帮助。

## 下载安装问题

### 浏览器提示"危险文件"或"不安全的下载"

这是浏览器对不常见下载文件的默认安全警告。开途软件经过安全签名认证，可以放心下载安装。在提示中选择"保留"或"仍然下载"即可。

### Windows 安装时弹出 SmartScreen 蓝色警告

Windows SmartScreen 可能会拦截新发布的软件。处理方法：点击 **"更多信息"**，然后点击 **"仍要运行"** 即可继续安装。

### macOS 提示"无法验证开发者"

右键点击安装包 → 选择 **"打开"** → 在弹出的对话框中点击 **"打开"** 确认。这样就可以正常安装了。

### Android 安装被拦截

不同品牌手机的处理方式不同，请参考对应品牌的安装指南。如果多次尝试仍然无法安装，推荐使用电脑 USB 辅助安装的方式（详见"通过电脑 USB 辅助安装"指南）。

## 更新问题

### 桌面版（Windows / Mac / Linux）如何更新？

开途桌面版支持应用内自动更新。收到更新提示时点击更新，更新完成后重启应用即可。

### Android 版如何更新？

应用内会提示有新版本可用，点击更新即可。更新流程与首次安装相同，可能会再次遇到安装拦截提示，按照首次安装时的方式处理即可。

### iOS 版如何更新？

iOS 版通过 App Store 自动更新。也可以打开 App Store 手动检查更新。

## 账号相关问题

### 忘记账号了怎么办？

开途账号就是你注册时使用的邮箱地址。在登录页面输入邮箱，选择"邮箱验证码登录"，输入收到的验证码即可重新登录，不需要记住密码。

### 一个账号可以在几台设备上用？

一个账号最多可以同时在 5 台设备上使用，支持不同平台混用（比如手机 + 电脑 + 平板）。

## 其他问题

### 应用闪退

请先确认已更新到最新版本。如果最新版本仍然闪退，请通过应用内反馈功能提交反馈，或联系客服获取帮助。

### 需要更多帮助？

如果以上方法都不能解决你的问题，请联系客服。详细的联系方式请参考"联系客服与获取帮助"页面。提交反馈时如果能通过应用内的反馈功能提交，会自动附带诊断日志，方便技术人员更快定位问题。
