# 开途使用指南

## 连接和断开

1. 打开开途应用，使用邮箱登录你的账号。
2. 登录后进入主界面，点击中间的**连接按钮**即可开始连接。
3. 连接成功后，手机状态栏或电脑系统托盘会显示 VPN 图标，表示开途正在运行。
4. 需要断开时，再次点击连接按钮即可断开。

## 路由模式

开途提供两种路由模式，可以在应用主界面直接切换。切换后会自动重新连接。

### 全局模式

所有网络流量都通过开途加速。适合以下场景：

- 需要全程使用海外网络
- 玩海外游戏
- 对隐私要求较高，希望所有流量都加密

### 智能模式（推荐日常使用）

自动识别国内和海外流量，分别处理：

- **国内网站和应用**：直接连接，不经过开途，保持原有速度
- **海外网站和应用**：自动通过开途加速

智能模式的好处是访问国内网站时速度不受影响，同时海外网站也能正常加速。推荐大多数用户日常使用这个模式。

## 节点选择

- 开途会默认选择最优节点，大多数情况下无需手动切换。
- 如果需要切换节点，在主界面点击当前节点名称即可查看可用节点列表。
- 节点列表会显示延迟信息，选择延迟较低的节点通常体验更好。

## 多设备使用

- 一个账号最多可以同时在 **5 台设备**上使用。
- 支持跨平台同时在线，比如手机、电脑、平板可以同时连接。
- 所有设备共享同一个订阅套餐，无需额外购买。
- 如果超过 5 台设备同时连接，最早登录的设备会被自动断开。

## 日常使用建议

- **推荐使用智能模式**：既能顺畅访问海外网站，又不影响国内网站的速度和体验。
- **手机端设置电池优化白名单**：将开途加入电池优化白名单（不限制后台运行），避免系统在后台自动关闭开途导致连接中断。设置路径一般在"设置" → "电池" → "应用电池管理"中。
- **开机自启**：开途支持开机自动启动功能，可以在应用的"设置"中开启，这样每次开机后无需手动打开应用。
- **保持应用更新**：建议及时更新到最新版本，新版本通常会优化连接速度和稳定性。
