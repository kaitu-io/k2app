# v0.3.19

Release Date: 2026-01-19

## New Features
- Domain search engine with LRU cache for improved rule matching performance
- Binary search lookup and domain index file builder for faster rule processing
- Smux buffer size negotiation for iOS memory optimization
- Memory pressure monitoring and reduced Go memory limit for iOS
- FileLogger for iOS extension debugging with Console.app visibility
- LeakDetector for goroutine baseline tracking with spike detection and stack traces
- Route diagnosis frontend integration

## Bug Fixes
- Windows: Skip ghost routes without valid adapter
- Windows: Skip third-party VPN adapters in primary interface selection
- Android: Fix unix/linux build constraints exclusion
- Android: Add missing offset parameter to fdbased.Open
- Mobile: Inject window._k2 for webapp stores compatibility
- Route diagnosis bypasses VPN and uses correct node identifiers
- Prevent raw error messages from being displayed to users

## Improvements
- Vendor tun2socks v2.6.0 as internal package
- iOS TUN buffer pool with blocking read implementation
- Enforce Single Source of Truth for tunnel selection in dashboard
- Refactor to use unique TUN IPs and exact matching for conflict detection
- Remove conflicting TUN IPs from third-party VPNs before setup
- Add golangci-lint configuration and goroutine patterns documentation
