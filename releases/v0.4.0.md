v0.4.0 是一次重大版本升级。全新自研拥塞控制协议 **k2cc** 正式上线——针对高丢包、高延迟的受干扰网络深度优化，即使在严重干扰环境下也能保持接近无损的传输速度。同时带来 iOS/Android 移动端支持、Linux 桌面端、全新官网、以及大量稳定性改进。

## New Features
- **k2cc 拥塞控制协议**: 自研拥塞控制算法，在高丢包、高延迟的严重干扰环境下保持接近无损的传输速度
- **iOS 和 Android 移动端**: 全新移动端客户端，支持 iOS Network Extension 和 Android VPN Service
- **Linux 桌面端**: 新增 Linux 平台支持（tar.gz 打包），一行命令安装
- **实时状态推送**: 连接状态从定时查询升级为 SSE 实时推送，操作响应更即时
- **Beta 测试通道**: 新增 beta 更新通道，提前体验新功能。Beta 模式自动收集诊断日志
- **Android USB 安装助手**: ADB 一键安装，支持自动设备检测和品牌适配指引
- **反馈系统**: 应用内反馈按钮 + 日志自动上传 + 工单追踪
- **k2 CLI**: 命令行工具支持 `k2 up/down/status`，可通过 PATH 直接调用
- **在线客服**: 官网集成 AI 客服 + 人工转接
- **官网全面重构**: 全新首页动效、下载页双 CDN、K2 协议文档、帮助中心

## Bug Fixes
- 修复弱网环境下连接异常断开后未能及时重连的问题
- 修复 QUIC stream slot 泄漏导致连接数耗尽的问题
- 修复 macOS DNS 崩溃后无法恢复的问题
- 修复 Windows 退出时双重断开崩溃
- 修复 iOS Network Extension 进程被杀后未能自动重连
- 修复 iOS 内存压力下 NE 被系统终止的问题
- 修复 Android 亮色模式状态栏显示异常
- 修复页面偶尔空白的显示问题
- 修复视口缩放时布局异常
- 修复 Windows 升级期间短暂断网
- 修复登录后旧 401 响应清除新 token 的竞态问题
- 修复 VPN 连接/断开按钮的双击竞态

## Improvements
- **连接稳定性**: DNS 容错增强 + 网络变化感知（netCoordinator）+ 自动重连探测
- **抗封锁能力**: 协议伪装增强 + antiblock Happy Eyeballs 多 CDN 并发探测
- **统一日志系统**: 全平台统一日志格式、轮转策略（20MB/3 备份/7 天）、stderr 重定向
- **错误分类**: 三方错误分类（客户端/网络/服务端/目标站点），错误提示更清晰
- **macOS 安装包瘦身**: universal binary，体积减少约 35%
- **Windows 签名**: macOS 交叉编译 + Authenticode 签名
- **iOS App Store 合规**: 隐私清单、账户删除、购买引导
- **路由规则升级**: 规则引擎 v0.6.1，智能分流更精准
- **内存优化**: iOS NE 内存诊断 + Go 内存限制 + smux 缓冲区协商

## Breaking Changes
- **k2arc 更名为 k2cc**: 拥塞控制协议品牌名统一为 k2cc（避免与业余无线电术语冲突）
