## New Features
- User hard delete functionality for admin
- HTTP POST /action/{action} endpoint for sync calls
- Desktop app shown in Dock and taskbar
- Viewport scaling for HiDPI displays

## Bug Fixes
- Windows NSIS uninstaller signing
- Desktop window sizing on HiDPI displays
- VPN stops on all exit methods (Dock, Cmd+Q)

## Improvements
- DNS handling simplified (TUN layer intercepts all port 53 traffic)
- VPN route configuration optimized (interface routes instead of gateway routes)
- macOS DNS handling extremely simplified
- Project documentation and build system refactored
- Tauri 2 window/tray/i18n system implemented
- Layered reconnection strategy implemented
