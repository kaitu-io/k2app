## New Features
- MPTCP (Multipath TCP) enhanced connection and DNS handling
- Comprehensive E2E test suite for multipath
- Windows E2E test suite and cross-platform testing
- OpenWRT router integration with LuCI iframe
- Responsive layout system for OpenWRT router mode
- K2S legacy protocol backward compatibility
- Local proxy testing environment with QoS simulation

- Windows default gateway restoration after VPN disconnect
- DNS resolver configuration on macOS
- S3 region and ACL configuration for deployment
- Cross-platform binaries preserved during desktop builds
- TLS-only SNI router handling

- K2Config simplified to use single active_tunnel
- Node offline marking functionality added
- Protocol naming conventions standardized
- Build scripts integrated with S3 upload
