## New Features
- **Desktop Release Notes**: Auto-updater displays detailed release notes before installing updates
- **Unified Changelog System**: Single source of truth for version history with web dashboard list view
- **Initialization Status**: Display service initialization progress during startup
- **API Request Logging**: Added request logging for debugging API failures

## Bug Fixes
- Fixed Windows update notification not showing before installation begins
- Fixed socket leak from TUN reserved IP traffic (resolved 110K+ file descriptor exhaustion)
- Fixed service not starting after admin_reinstall_service on macOS
- Fixed Go service port restored to 1777
- Fixed desktop update mode skipping finish page run option
- Fixed SmartScreen submission URL display in deployment scripts

- Removed route diagnosis feature to simplify codebase (~850 lines)
