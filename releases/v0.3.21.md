## New Features
- **Desktop Release Notes**: Auto-updater displays detailed release notes before installing updates
- **Unified Changelog System**: Single source of truth for version history with web dashboard list view
- **Initialization Status**: Display service initialization progress during startup
- **API Request Logging**: Added request logging for debugging API failures
- Domain search engine with LRU cache for improved rule matching performance
- Binary search lookup and domain index file builder for faster rule processing
- Smux buffer size negotiation for iOS memory optimization
- Memory pressure monitoring and reduced Go memory limit for iOS
- FileLogger for iOS extension debugging with Console.app visibility
- LeakDetector for goroutine baseline tracking with spike detection and stack traces
- Route diagnosis frontend integration
