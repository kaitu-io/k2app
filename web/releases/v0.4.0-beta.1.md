v0.4.0 是一次重大版本升级。全新自研拥塞控制协议 **K2ARC** 正式上线——针对高丢包、高延迟的受干扰网络深度优化，即使在严重干扰环境下也能保持接近无损的传输速度，流媒体、视频会议、大文件下载均不受影响。同时带来更快的状态响应、更好的连接稳定性、全新的网站体验，以及大量细节优化。

## New Features
- **K2ARC 拥塞控制协议**: 自研拥塞控制算法，在高丢包、高延迟的严重干扰环境下保持接近无损的传输速度
- **实时状态推送**: 连接状态从定时查询升级为实时推送，操作响应更即时，断线感知更迅速
- **Beta 测试通道**: 新增 beta 更新通道，提前体验新功能。Beta 模式自动收集诊断日志，帮助我们更快定位问题
- **开发者模式**: 高级用户可在运行时切换日志级别，便于自助排查
- **K2 协议文档**: 官网新增 [K2 协议技术文档](/k2/)专区，中英文对照
- **家长指引页**: 新增[家长指引](/guide/parents)页面，方便学校推荐的家长了解产品
- **反馈按钮**: 应用内新增可拖拽的反馈悬浮按钮，随时提交使用反馈
- **在线客服**: 官网集成在线客服，获取帮助更便捷

## Bug Fixes
- 修复弱网环境下连接异常断开后未能及时重连的问题
- 修复 macOS 12.5 及更早版本上输入框无法点击聚焦的问题
- 修复页面偶尔空白的显示问题
- 修复视口缩放时布局异常
- 修复特定情况下反复弹出更新提示的问题
- 修复错误状态下 VPN 按钮行为不合理的问题
- 修复 Windows 升级期间短暂断网的问题

## Improvements
- **连接稳定性**: DNS 容错增强，减少因 DNS 问题导致的连接失败
- **路由规则升级**: 规则引擎更新至 v0.6.1，智能分流更精准
- **错误提示优化**: 连接错误展示更清晰，不再出现难以理解的技术信息
- **抗干扰增强**: 提升协议伪装能力，改善受干扰网络环境下的可用性
- **内存占用降低**: 优化连接层内存使用
- **窗口行为优化**: 关闭窗口改为最小化到托盘，支持开机最小化启动
- **macOS 安装包瘦身**: 安装包体积减少约 35%
- **升级更平滑**: macOS 和 Windows 升级过程优化
- **官网首页**: 全新品牌标语，导航重新设计，页面加载更快
- **邀请页面**: 简化页面布局，视觉更整洁
- **日志系统**: 全新结构化日志架构，排障信息更完整
