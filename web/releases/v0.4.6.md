**开途 0.4.6 重点更新**：新增**抗封锁自动中继**（控制通道被封锁时也能自动绕行连上，专治「新装或换了网络后第一次连不上」）、iOS 版上线**App 内订阅**、修复**休眠 / 后台回收后「假连接」**顽疾，以及一批连接可用性与稳定性改进。

路由器安装命令：`wget -qO- https://kaitu.io/i/k2r | sudo sh`（本版新增 MIPS 架构，覆盖更多老款路由器）。

## New Features

- **抗封锁自动中继**：获取节点列表、分流规则用的「控制通道」一旦被封锁，App 会自动改用健康节点做中继完成握手，并内置一份「保底」节点种子。即使在最恶劣的网络环境下，**新安装、或者换了 Wi-Fi / 数据网络后的第一次连接**也能顺利建立，不再卡在「一直转圈连不上」——这是最难自救的冷启动场景
- **iOS App 内订阅**：iPhone / iPad 现在可以直接在 App 内用 Apple 账户订阅开通会员，支持「恢复购买」；会员中心重新设计，续订、到期时间、订阅管理一目了然
- **住宅 IP 标识**：节点列表为住宅 IP 线路加上专属标记，挑选不易被目标网站风控的线路更直观

## Bug Fixes

- **修复休眠 / 后台内存回收后「显示已连接却上不了网」**：手机锁屏、系统 Doze、内存紧张回收隧道之后，App 现在能准确识别并在数十秒内自动重连自愈，不再需要手动断开重连（安卓、iOS 均已真机验证）
- **修复部分出口「缺规则包」导致网页 504 打不开**：地理位置检测走隧道被污染时，会误判所在国家、给某些出口（例如日本）下发了缺失的分流规则包，导致部分网站连不上；现已锚定正确区域，缺包不再发生
- **修复注销账号后用同一邮箱重新登录报错**：删除账号会正确释放邮箱占用，用原邮箱重新注册 / 登录不再报 500
- **修复偶发的规则库更新失败**：剔除已失效的下载镜像、去掉过于严格的校验，规则库在弱网 / CDN 抖动下下载更稳
- **过滤第三方浏览器注入脚本引发的错误噪音**：部分国产浏览器（UC / 鸿蒙内核）注入脚本会污染错误上报，已过滤，不影响正常使用

## Improvements

- **智能选服更聪明**：优化节点推荐算法（按流量周期的时间进度动态加权），把更贴近真实可用额度的线路排在前面——周期初期宽松、临近月底自动避开接近流量上限的节点
- **开途路由器支持更多老款机型**：新增 MIPS / MIPSLE 架构固件，更多老款硬路由 / 软路由可一键安装
- **iOS 合规调整**：按 App Store 要求，iOS 版仅保留 App 内订阅入口，移除其它支付方式
- **系统要求更新**：macOS 最低支持版本提升至 12.0（Monterey）及以上
