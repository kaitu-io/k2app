**开途 0.4.5 重点更新**：新增「按应用排除」功能（让微信、银行、12306 等怕 VPN IP 的应用走直连）、推出**密码登录**（不再只能等邮箱验证码）、推出**开途路由器**完整支持、以及大量稳定性与体验改进。

路由器安装命令：`wget -qO- https://kaitu.io/i/k2r | sudo sh`（自动识别 ARM64 / x86_64 / ARMv7 三种架构）。

## New Features

- **按应用排除**：在「Dashboard → 高级设置 → 不走代理的应用」可以把特定应用排除在 VPN 之外。常见场景：银行 App、微信、12306、网游等会因 VPN IP 被风控的应用
- **应用候选自动列出**：桌面端（Mac / Windows / Linux）列出正在运行的应用；安卓列出已安装应用，并按地区自动预置常用候选（中国大陆预置微信 / 支付宝 / 12306 / 主流银行 App）
- **隐私守护**：被排除的应用名永远不会出现在日志、反馈包、错误上报里
- **连接中锁定高级设置**：连接 VPN 时高级设置变为只读，断开后才能修改 —— 解决了「我改了设置好像没生效」的困惑
- **密码登录**：电脑 / 手机 / 网站全部支持邮箱 + 密码登录，不再只能等验证码邮件。登录界面新增「密码」标签
- **账户安全页**：网站新增 `/account/security`、桌面与手机 App 在「我的」里新增「密码」入口，可设置或修改密码；强度计实时反馈
- **改密码自动发邮件提醒**：每次成功设置或修改密码后会收到通知邮件，帮你第一时间发现账号异常
- **开途路由器（k2r）正式发布**：硬路由 / 软路由 / Linux 单板机一键安装，TPROXY 全局透明代理，断电重连、配置持久化、固件 OTA 升级齐全
- **路由器登录提示更清晰**：当套餐不支持路由器、或者用错设备类型（手机账号当作路由器登录等）时，会给出明确指引（"套餐不支持路由器请升级" / "登录身份不符请重新登录"），不再出现模糊错误或无声失败

## Bug Fixes

- **修复 App Bypass 在 Mac / Windows / Linux 上有时漏排除**：进程归属在多种系统场景下被静默丢掉（Windows 上的 UDP / QUIC / 网游、Linux 上刚启动的进程、Mac 上 Chrome 等多进程应用），现在都能准确识别
- **修复 iOS 旧版（16.0–16.3）打开网站白屏**：兼容旧版 Safari 语法，老款 iPhone / iPad 可正常访问
- **修复安卓全屏切换后底部留白**：边到边沉浸态调整后，界面正确铺满屏幕
- **修复 iPad 横屏被误识别为电脑端**：现按屏幕短边判定，平板横竖屏布局都正确
- **修复长时间连接后出现「连着但用不了」**：服务端 / 客户端通信层加固，长会话下不再卡死
- **修复密码登录有时「点了没反应」**：网站密码登录成功后正确保存登录态，不再静默丢失
- **修复路由器在某些场景下崩溃 / 启动失败**：合并了在用户现场捕获的 6 项路由器稳定性问题（保留启动前的 iptables、增加 iptables 锁等待、防止守护进程无限重启、修复 DNS 泄漏、安装脚本预检内核支持等）
- **修复部分国内品牌手机自带浏览器白屏**（华为 / 小米 / OPPO / vivo 的旧 WebView）
- **修复 App 内点删除等确认按钮没反应**：从底层弹窗改为应用内确认弹窗
- **修复偶发的规则更新失败**：剔除两个已失效的镜像，规则下载更稳定

## Improvements

- **App Bypass 列表打开更快**：进入页面或重新扫描时不再每次重新查询系统进程，并发刷新自动合并
- **规则库每小时自动刷新**：与是否连接 VPN 解耦，规则始终保持最新
- **Mac 端安装更简单**：去掉了系统扩展（System Extension）安装步骤，统一走守护进程模式，权限提示更少
- **抗 DNS 污染加强**：访问国内服务改用更可靠的 HTTPDNS 锚点解析
- **安卓 11 及以上版本兼容**：补齐应用列表权限声明，App Bypass 候选不再为空
- **iOS App Store 版本号统一**：营销版本号与桌面 / 安卓三段对齐，方便对照
