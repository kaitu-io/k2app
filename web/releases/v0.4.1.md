v0.4.1 修复了多个影响日常使用的重要问题，特别是设备被无故踢下线的 bug，以及移动端冷启动后隧道身份丢失的问题。同时改进了在线客服的智能对话体验，并优化了多个平台的稳定性。

## New Features
- **智能客服增强**: AI 客服新增交互式欢迎按钮、视频/语音通话请求自动转人工、账户相关问题自动转人工
- **官网帮助中心**: 全新设计的帮助页面，包含视频教程、常见问题、SEO 优化
- **Linux tar.gz 更新器**: Linux 从 AppImage 切换到 tar.gz 打包，更新包体积从 85MB 降至 ~8MB
- **三方错误分类**: 引擎错误区分客户端/网络/服务端/目标站点四类，target 类错误不影响 VPN 状态
- **诊断日志系统**: 新增结构化诊断日志，支持心跳、DNS、传输层、代理拨号等多维度快速定位

## Bug Fixes
- **修复设备无故被踢下线**: 两台不同 Mac 可能生成相同的硬件标识符，导致设备在不同账号间反复转移。现已改用随机标识符 + 本地持久化，彻底消除碰撞问题
- **修复移动端冷启动后隧道身份丢失**: iOS/Android 冷启动后未能恢复隧道身份，导致连接失败
- **修复登录后旧 401 响应清除新 token**: 竞态条件下旧请求的 401 响应可能清除刚登录获取的新 token
- **修复更新通知遮挡移动端安全区域**: UpdateNotification 横幅未适配 iOS 刘海屏安全区域
- **修复 Windows 退出时双重断开崩溃**: Windows 退出时 stop_vpn 被调用两次导致崩溃
- **修复 Android 亮色模式状态栏**: Android 设备系统亮色模式下状态栏显示为白色
- **修复 iOS NE 进程被杀后自动重连**: iOS 系统终止 Network Extension 后现在能自动恢复连接
- **修复 Linux FUSE 限制下的 k2 二进制暂存路径**: pkexec 无法访问 FUSE 挂载点

## Improvements
- **抗封锁能力**: antiblock Happy Eyeballs 策略，6 个 CDN 镜像并发探测，避免中国区 10 秒超时
- **iOS 内存优化**: Network Extension 内存诊断和 DNS 并发控制
- **构建流程优化**: macOS/Windows 签名和发布流程改进，S3 上传与构建分离防止误发布

## Breaking Changes
- **首次升级需要重新登录**: 设备标识符生成方式变更，首次打开新版本后需要重新登录一次。此后不再需要
